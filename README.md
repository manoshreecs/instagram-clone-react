# 📸 Instagram Clone (React)

A **modern Instagram-inspired frontend application** built with **React**, **Bootstrap 5**, and **React Router**.  
The project replicates key Instagram interface components such as **Stories, Feed Posts, Profile, and User Suggestions** using **mock JSON data** to simulate a social media environment.

This project demonstrates **component-based architecture, state management, and responsive UI design**, making it suitable for learning and showcasing **frontend development skills**.

> ⚠️ **Note:** This project is a frontend simulation and does not include backend authentication, database storage, or real-time updates.

---

## ✨ Features

### 🏠 Home Feed
- Displays posts with **images, captions, and like counts**
- Clean card-based layout similar to Instagram

### 📖 Stories
- Horizontal **story carousel**
- Click to view stories
- Navigate between stories

### 👤 Profile Page
- Displays **user profile**
- Shows **followers list**
- Organized profile layout

### ➕ Suggestions
- Displays **recommended users**
- **Follow / Unfollow functionality**

### 📱 Responsive Design
- Optimized for **desktop, tablet, and mobile screens**

### 🎨 Modern UI
- Built using **Bootstrap 5**
- Clean and minimal design inspired by Instagram

---

## 🛠️ Technologies Used

- **React** – Component-based UI development  
- **React Router** – Client-side routing  
- **Bootstrap 5** – Responsive styling and layout  
- **React Context API** – Global state management  
- **Vite** – Fast development and build tool  
- **Mock JSON Data** – Used to simulate posts, stories, and user suggestions

---

## 📚 Learning Outcomes

This project demonstrates:

- React **component-based architecture**
- **State management using Context API**
- **Routing with React Router**
- Building **responsive layouts**
- Handling **mock data for UI simulation**
- Structuring a **scalable frontend project**
